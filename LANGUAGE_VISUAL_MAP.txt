│  6️⃣  React перерендеривает компоненты                                     │
│      └─ все t('key') возвращают русские значения                          │
│                                                                            │

┌─ 🎯 КЛЮЧЕВЫЕ ФУНКЦИИ ─────────────────────────────────────────────────────┐
│                                                                            │
│  ▸ withLang(path: string)                                                 │
│    Автоматически добавляет языковой префикс к URL                         │
│    Пример: withLang('/services') → '/de/services' или '/ru/services'     │
│                                                                            │
│  ▸ t(key: string)                                                         │
│    Получает перевод по ключу                                              │
│    Пример: t('nav.home') → 'Home' (de) или 'Главная' (ru)               │
│                                                                            │
│  ▸ getLanguageFromURL()                                                   │
│    Определяет язык из текущего URL                                        │
│    Пример: /ru/about → 'ru', /de/services → 'de'                         │
│                                                                            │
